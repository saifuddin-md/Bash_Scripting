# Bash Scripting

**A collection of practical Bash scripts for Linux administration, automation, monitoring, and day-to-day DevOps tasks.**

**This repository includes scripts for:**
- Disk, memory, and system health monitoring
- Log cleanup and alerting
- Service auto-restart automation
- Kubernetes utility scripts
- Jenkins, Docker, Nexus, and SonarQube setup helpers

*These scripts are built to simplify repetitive operational tasks and improve server management efficiency.*
